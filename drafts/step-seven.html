Step 7 - Git Version Control

Published: Fri 31 May 2024 | Tags: programming, pelican, pelican tutorial, python,

The two main reasons people use the internet are to be entertained and seek information. You will upload your website for everybody to see and read.

The Python syntax is super easy to read and follow. Pelican is also very simple and ideal for you as an introduction to programming. This tutorial will help you set up a blog, a website or both depending on what need is. And host it on GitHub for the world to see.

Create a GitHub Account

  1. Open your browser and paste https://github.com
  2. Click the sign up button
  3. Follow the instructions. When you choose your account know it will be part of the url for your site, like this https://github.com/YOUR-ACCOUNT-NAME/repository-name. If you have a domaian name you can connect that to the GitHub site, and then the account name will not be visible.
  4. When asked to "Unlock advanced features with GitHub Team or continue with a free plan for the basics.", click continue for free.
  5. Click on the icon in the top right corner.
  6. Towards the bottom click on Settings.
  7. On the left, click on Password and authentication.
  8. Set up Passkey and Two-factor authentication.

Create a GitHub Repository

  1. On the left, click the Create repository button.
  2. Add the repository name, for example yogurt-journey.
  3. Click the Create repository button at the bottom.

Uploading Output Folder to GitHub

  1. Open command line. cd and cd .. into the output folder.
  2. Type git init and hit return. Wait for it to run the set up.
  3. Follow the instructions from GitHub for commandline.
  4. Type git init and hit return.
  5. Type git commit -m "first commit" and hit return.
  6. Type git branch -M main and hit return.
